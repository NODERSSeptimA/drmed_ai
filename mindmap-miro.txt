MedAI — AI-ассистент для врачей
	Продукт
		Ценность
			Быстрое заполнение истории болезни голосом
			Снижение рутины врача на 60-70%
			Готовый Word-документ для печати
			Единый формат документации по клинике
		Пользователь
			Врач (любая специализация)
			Первый фокус — психиатрия
		Первая функция (MVP)
			Голосовой диалог с AI по шаблону истории болезни
			AI задает вопросы — врач отвечает голосом
			Автозаполнение всех разделов
			Экспорт в Word по шаблону клиники
	Пользовательский сценарий
		1. Вход в систему
			Логин врача
			Привязка к клинике и специализации
		2. Выбор пациента
			Создать нового пациента
			Выбрать существующего
		3. Начать голосовую сессию
			Нажать кнопку "Голос"
			AI приветствует и задает первый вопрос
		4. Диалог с AI
			AI задает вопрос по текущему разделу
			Врач отвечает голосом
			AI извлекает данные и заполняет поля
			AI задает уточняющие вопросы если нужно
			Переход к следующему разделу
		5. Проверка результата
			Просмотр заполненной истории
			Ручная правка если нужно
		6. Экспорт
			Генерация Word-файла
			Скачивание / печать
	Технический стек
		Frontend
			Next.js 15 (App Router)
			TypeScript
			Tailwind CSS
			React-компоненты
		Backend
			Next.js API Routes
			WebSocket (socket.io)
			Prisma ORM
		База данных
			PostgreSQL
			Модели: User, Clinic, Patient, MedicalHistory, Template, VoiceSession
		AI сервисы
			OpenAI Whisper API (Speech-to-Text)
			GPT-4o (диалог и извлечение данных)
			OpenAI TTS (озвучка вопросов AI — опционально)
		Генерация документов
			Библиотека docx (npm)
			Word-шаблоны с плейсхолдерами
		Деплой
			Vercel (frontend + API)
			Railway / Supabase (PostgreSQL)
			S3 / R2 (хранение аудио)
	Архитектура
		Система шаблонов
			JSON-схема для каждой специализации
			Разделы и поля с типами данных
			Вопросы AI для каждого поля
			Уточняющие вопросы и ветвление
			Привязка к клинике
		Conversation Engine
			State machine диалога
			Состояния: Idle → Recording → Processing → Extracting → Reviewing
			System prompt с шаблоном + заполненными данными
			Structured output от LLM (JSON)
		Голосовой pipeline
			Web Audio API — захват микрофона
			Отправка аудио на сервер (WebSocket)
			Whisper API — транскрипция
			GPT-4o — извлечение данных + следующий вопрос
			TTS — озвучка ответа AI (опционально)
		Word-генератор
			Шаблон .docx с плейсхолдерами
			Подстановка данных из БД
			Форматирование по стандартам клиники
	Экраны приложения (MVP)
		Страница входа
			Логин / пароль
		Главная (список пациентов)
			Список пациентов на сегодня
			Кнопка "Новый пациент"
			Быстрый доступ к AI-сессии
		Голосовая сессия (ключевой экран)
			Левая панель — разделы шаблона с прогрессом
			Центр — чат с AI + кнопка микрофона
			Правая панель — превью истории болезни
		Просмотр истории болезни
			Все разделы с данными
			Редактирование
			Экспорт в Word
	Шаблон «Осмотр психиатра» (Династия)
		1. Данные пациента
			ФИО пациента
			Дата рождения
		2. Основания для осмотра
			По данным наблюдения, мед. документов
		3. Жалобы
		4. Анамнез
			Диспансерное наблюдение (психиатр, нарколог, др.)
			Нейроинфекции, проф. вредности
			Пагубное употребление ПАВ (алкоголь, наркотики, лекарства)
			Психотравмирующие ситуации
			Суицидальные попытки в прошлом
			Эпизоды нарушения сознания, судороги, ЧМТ
			Хронические заболевания и фармакотерапия
			Дополнения
		5. Психический статус
			Сознание и ориентировка
			Ответы на вопросы
			Контакт с больным (продуктивность)
			Речь
			Внимание
			Данные наблюдения за пациентом
			Эмоциональные нарушения
			Нарушение памяти / интеллекта
			Нарушения мышления (навязчивые, сверхценные, бредовые идеи)
			Нарушения ощущений — количественные (анестезия, гипо-, гиперстезия)
			Нарушения ощущений — качественные (синестезия, парестезия)
			Нарушения восприятия (иллюзии, галлюцинации, сенестопатии, деперсонализация)
			Двигательно-волевые нарушения (абулия, импульсивность, негативизм)
			Суицидальные намерения
			Самооценка психического состояния
		6. Сомато-неврологический статус
		7. Динамика состояния за время наблюдения
		8. Заключение
			Особенности психического статуса → симптомы
			Уровень нарушений: психотический / дефицитарный
			Тип реагирования: эндогенный / экзогенно-органический / психогенный / смешанный
		9. Диагноз
			Текст диагноза
			Код МКБ-10 (F___)
		10. План обследования
			Консультации специалистов, ЭКГ, УЗИ, ФГ, ОАМ, ОАК, глюкоза, биохимия
		11. План лечения
		12. Повторный осмотр
			Дата повторного осмотра
			Врач-психиатр, подпись, дата
	Фазы реализации
		Фаза 1 — Фундамент (1-2 нед)
			Next.js + TypeScript + Tailwind
			PostgreSQL + Prisma
			UI-компоненты
			Авторизация
		Фаза 2 — AI Ядро (2-3 нед)
			Whisper API интеграция
			Conversation Engine
			Шаблон психиатрии
			Извлечение данных из речи
		Фаза 3 — Интерфейс сессии (1-2 нед)
			Экран голосовой сессии
			Захват аудио
			Прогресс заполнения
			Realtime превью
		Фаза 4 — Word экспорт (1 нед)
			Шаблон Word для психиатрии
			Генерация .docx
			Скачивание файла
		Фаза 5 — Полировка (1 нед)
			Редактирование истории
			Обработка ошибок
			Адаптив под планшеты
	Стоимость AI на 1 историю
		Whisper STT — ~$0.06 (10 мин)
		GPT-4o — ~$0.15-0.30 (15-20 вызовов)
		TTS — ~$0.09 (опционально)
		Итого — ~$0.30-0.45
